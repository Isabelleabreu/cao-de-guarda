## Descrição

Projeto desenvolvido em sala de aula. Inicialmente, realizamos a preparação do ambiente e testamos a conexão com Wi-Fi utilizando nosso smartphone. Em seguida, partimos para a montagem física com protoboard, jumpers e LEDs.

Com base no código utilizado para testar a conexão Wi-Fi, criamos o “espião” de rede que monitorava o sinal, utilizando uma API no projeto. Verificávamos se a conexão retornava o status 200; caso positivo, o LED acendia. Caso contrário, o LED apagava e retornava **false**.
